# USE NODE >=14.15.3 & NPM

# command

## Install

```shell script
$ npm ci
```

## Watch

#### watch when development

```shell script
$ npm start
```

#### watch when production

```shell script
$ npm run start-pro
```

## Build

#### build when development

```shell script
$ npm run build-dev
```

#### build when production

```shell script
$ npm run build
```

## Build & Watch

```shell script
$ npm run serve
```

# HOW TO

1. git clone
2. check out `develop` brunch
3. create `your name` brunch from `develop`
4. `npm ci`
5. `npm start`
